Devdocs ticket

The procedures in this article are the most requested by other departments.

  • Create a devdocs ticket to request changes needed in the dev center.

  • Contact devdocs if you have any questions.

Devdocs pull request

Create a PR to make changes in the source file.

  • Ensure you are in the bigcommerce/docs repository.

  • The devdocs team will review your PR.

Preview changes

Use npm to create an instance to view your changes.

  • Use nvm version 18+.

  • We recommend using visual studio.

Beta access

NOTE: Ensure you have product manager approval.

  • Request beta access for merchants.

  • You will also need first and last name and their github email address before requesting access.

  • Fill out the Beta Project and Customer List.
